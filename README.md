# The Algorithm Will Score You
An interactive Twine experience that simulates how predictive policing algorithms assign risk scores to individuals based on demographic and socioeconomic data.

## About
Players answer 7 questions (neighborhood, age, employment status, and more) and receive a risk score calculated the same way real tools like COMPAS and PredPol work. The experience is designed to make algorithmic bias feel personal.

## Topics Covered
- What predictive policing is and how it works
- Racial and socioeconomic disparities in risk scoring
- Real-world consequences: bail, patrol targeting, watch lists
- Who builds and profits from these tools
- Resources for learning more and taking action
